Introduction
============

|var-project| is a graphical tool to visualize the current DDS system.

Overview
--------

- Visualize the DDS entities in your system
- Tester to simulate DDS data flow
- Listener to monitor DDS topics
- Inspect the QoS settings of each entity
- Cyclone DDS Statistics integration
- Cross-platform (Linux, Windows, macOS)
- and more...
