Throughput¶

The Throughput example allows the measurement of data throughput when receiving samples from a publisher.

Design¶

The Throughput example consists of two units:

  • Publisher: Sends samples at a specified size and rate.

  • Subscriber: Receives samples and outputs statistics about throughput

Scenario¶

The publisher sends the samples. You can specify a payload size in bytes, and whether to send data in bursts. Unless a time-out is specified, the publisher sends data forever.

Configuration:

  • payloadSize: The size of the payload in bytes.

  • burstInterval: The time interval between each burst in ms.

  • burstSize: The number of samples to send each burst.

  • timeOut: The number of seconds the publisher should run for (0=infinite).

  • partitionName: The name of the partition.

The subscriber receives the data and outputs:

  • The total amount of data received.

  • The data-rate in bytes-per-second.

  • Whether any samples were received out-of-order.

You can specify the maximum number of cycles. When the maximum is reached, the subscriber terminates and outputs the totals and averages.

The subscriber executable measures:

  • transferred: the total amount of data transferred in bytes.

  • outOfOrder: the number of samples that were received out of order.

  • The transfer rate: the data transfer rate in bytes per second.

  • The subscriber also calculates statistics on these values over a configurable number of cycles.

Configuration:

  • maxCycles: The number of times to output statistics before terminating.

  • pollingDelay

  • partitionName: The name of the partition.

Running the example¶

To avoid mixing the output, run the publisher and subscriber in separate terminals.

  1. Open 2 terminals.

  2. In the first terminal start publisher by running publisher.

    Publisher usage (parameters must be supplied in order):

    ./publisher [payloadSize (bytes)] [burstInterval (ms)] [burstSize (samples)] [timeOut (seconds)] [partitionName]

    Defaults:

    ./publisher 8192 0 1 0 "Throughput example"

  3. In the second terminal start subscriber by running subscriber.

    Subscriber usage (parameters must be supplied in order):

    ./subscriber [maxCycles (0=infinite)] [pollingDelay (ms, 0 = event based)] [partitionName]

    Defaults:

    ./subscriber 0 0 "Throughput example"

  4. To achieve optimal performance, set the CPU affinity so that the publisher and subscriber run on separate CPU cores and use real-time scheduling:

Publisher usage:

taskset -c 0 chrt -f 80 ./publisher [payloadSize (bytes)] [burstInterval (ms)] [burstSize (samples)] [timeOut (seconds)] [partitionName]

Subscriber usage:

taskset -c 1 chrt -f 80 ./subscriber [maxCycles (0 = infinite)] [pollingDelay (ms, 0 = event based)] [partitionName]

Publisher usage:

START /affinity 1 /high cmd /k "publisher.exe" [payloadSize (bytes)] [burstInterval (ms)] [burstSize (samples)] [timeOut (seconds)] [partitionName]

Subscriber usage:

START /affinity 2 /high cmd /k "subscriber.exe" [maxCycles (0 = infinite)] [pollingDelay (ms, 0 = event based)] [partitionName]

137{
138  dds_entity_t participant;
139  dds_entity_t topic;
140  dds_entity_t publisher;

145  /* A domain participant is created for the default domain. */
146  participant = dds_create_participant (DDS_DOMAIN_DEFAULT, NULL, NULL);

150  /* Enable write batching */

154  tQos = dds_create_qos ();
155  dds_qset_reliability (tQos, DDS_RELIABILITY_RELIABLE, DDS_SECS (10));
156  dds_qset_history (tQos, DDS_HISTORY_KEEP_ALL, 0);
157  dds_qset_resource_limits (tQos, MAX_SAMPLES, DDS_LENGTH_UNLIMITED, DDS_LENGTH_UNLIMITED);
158  topic = dds_create_topic (participant, &ThroughputModule_DataType_desc, "Throughput", tQos, NULL);
159  if (topic < 0)
160    DDS_FATAL("dds_create_topic: %s\n", dds_strretcode(-topic));
161
162  /* A publisher is created on the domain participant. */
163  pubQos = dds_create_qos ();
164  pubParts[0] = partitionName;
165  dds_qset_partition (pubQos, 1, pubParts);
166  publisher = dds_create_publisher (participant, pubQos, NULL);
167  if (publisher < 0)
168    DDS_FATAL("dds_create_publisher: %s\n", dds_strretcode(-publisher));
169  dds_delete_qos (pubQos);
170
171  /* A DataWriter is created on the publisher. */
172  *writer = dds_create_writer (publisher, topic, NULL, NULL);
173  if (*writer < 0)
174    DDS_FATAL("dds_create_writer: %s\n", dds_strretcode(-*writer));
175  dds_delete_qos (tQos);
176
177  return participant;
178}

207static void start_writing(
208    dds_entity_t writer,
209    ThroughputModule_DataType *sample,
210    int burstInterval,
211    uint32_t burstSize,
212    int timeOut)
213{
214  bool timedOut = false;
215  dds_time_t pubStart = dds_time ();
216  dds_time_t now;
217  dds_time_t deltaTv;
218  dds_return_t status;
219
220  if (!done)
221  {
222    dds_time_t burstStart = pubStart;
223    unsigned int burstCount = 0;
224
225    printf ("=== [Publisher]  Writing samples...\n");
226    fflush (stdout);
227
228    while (!done && !timedOut)
229    {
230      /* Write data until burst size has been reached */
231
232      if (burstCount < burstSize)
233      {
234        status = dds_write (writer, sample);
235        if (status == DDS_RETCODE_TIMEOUT)
236        {
237          timedOut = true;
238        }
239        else if (status < 0)
240        {
241          DDS_FATAL("dds_write: %s\n", dds_strretcode(-status));
242        }
243        else
244        {
245          sample->count++;
246          burstCount++;
247        }
248      }
249      else if (burstInterval)
250      {
251        /* Sleep until burst interval has passed */
252
253        dds_time_t time = dds_time ();
254        deltaTv = time - burstStart;
255        if (deltaTv < DDS_MSECS (burstInterval))
256        {
257          dds_write_flush (writer);
258          dds_sleepfor (DDS_MSECS (burstInterval) - deltaTv);
259        }
260        burstStart = dds_time ();
261        burstCount = 0;
262      }
263      else
264      {
265        burstCount = 0;
266      }
267
268      if (timeOut)
269      {
270        now = dds_time ();
271        deltaTv = now - pubStart;
272        if ((deltaTv) > DDS_SECS (timeOut))
273        {
274          timedOut = true;
275        }
276      }
277    }
278    dds_write_flush (writer);
279
280    printf ("=== [Publisher]  %s, %llu samples written.\n", done ? "Terminated" : "Timed out", (unsigned long long) sample->count);
281    fflush (stdout);
282  }
283}

155static int do_take (dds_entity_t reader)
156{
157  int samples_received;
158  dds_instance_handle_t ph = 0;
159  HandleEntry * current = NULL;
160
161  if (startTime == 0)
162  {
163    startTime = dds_time ();
164  }
165
166  /* Take samples and iterate through them */
167
168  samples_received = dds_take (reader, samples, info, MAX_SAMPLES, MAX_SAMPLES);
169  if (samples_received < 0)
170  {
171    DDS_FATAL("dds_take: %s\n", dds_strretcode(-samples_received));
172  }
173
174  for (int i = 0; !done && i < samples_received; i++)
175  {
176    if (info[i].valid_data)
177    {
178      ph = info[i].publication_handle;
179      current = retrieve_handle (imap, ph);
180      ThroughputModule_DataType * this_sample = &data[i];
181
182      if (current == NULL)
183      {
184        current = store_handle (imap, ph);
185        current->count = this_sample->count;
186      }
187
188      if (this_sample->count != current->count)
189      {
190        outOfOrder++;
191      }
192      current->count = this_sample->count + 1;
193
194      /* Add the sample payload size to the total received */
195
196      payloadSize = this_sample->payload._length;
197      total_bytes += payloadSize + 8;
198      total_samples++;
199    }
200  }
201  return samples_received;
202}

240static void process_samples(dds_entity_t reader, unsigned long long maxCycles)
241{
242  dds_return_t status;
243  unsigned long long prev_bytes = 0;
244  unsigned long long prev_samples = 0;
245  dds_attach_t wsresults[2];
246  dds_time_t deltaTv;
247  bool first_batch = true;
248  unsigned long cycles = 0;
249  double deltaTime = 0;
250  dds_time_t prev_time = 0;
251  dds_time_t time_now = 0;
252
253  while (!done && (maxCycles == 0 || cycles < maxCycles))
254  {
255    if (pollingDelay > 0)
256      dds_sleepfor (DDS_MSECS (pollingDelay));
257    else
258    {
259      status = dds_waitset_wait (waitSet, wsresults, sizeof(wsresults)/sizeof(wsresults[0]), DDS_MSECS(100));
260      if (status < 0)
261        DDS_FATAL("dds_waitset_wait: %s\n", dds_strretcode(-status));
262    }
263
264    if (pollingDelay >= 0)
265    {
266      while (do_take (reader))
267        ;
268    }
269
270    time_now = dds_time();
271    if (!first_batch)
272    {
273      deltaTv = time_now - prev_time;
274      deltaTime = (double) deltaTv / DDS_NSECS_IN_SEC;
275
276      if (deltaTime >= 1.0 && total_samples != prev_samples)
277      {
278        printf ("=== [Subscriber] %5.3f Payload size: %lu | Total received: %llu samples, %llu bytes | Out of order: %llu samples "
279                "Transfer rate: %.2lf samples/s, %.2lf Mbit/s\n",
280                deltaTime, payloadSize, total_samples, total_bytes, outOfOrder,
281                (deltaTime != 0.0) ? ((double)(total_samples - prev_samples) / deltaTime) : 0,
282                (deltaTime != 0.0) ? ((double)((total_bytes - prev_bytes) / BYTES_PER_SEC_TO_MEGABITS_PER_SEC) / deltaTime) : 0);
283        fflush (stdout);
284        cycles++;
285        prev_time = time_now;
286        prev_bytes = total_bytes;
287        prev_samples = total_samples;
288      }
289    }
290    else
291    {
292      prev_time = time_now;
293      first_batch = false;
294    }
295  }
296
297  /* Output totals and averages */
298  deltaTv = time_now - startTime;
299  deltaTime = (double) (deltaTv / DDS_NSECS_IN_SEC);
300  printf ("\nTotal received: %llu samples, %llu bytes\n", total_samples, total_bytes);
301  printf ("Out of order: %llu samples\n", outOfOrder);
302  printf ("Average transfer rate: %.2lf samples/s, ", (double)total_samples / deltaTime);
303  printf ("%.2lf Mbit/s\n", (double)(total_bytes / BYTES_PER_SEC_TO_MEGABITS_PER_SEC) / deltaTime);
304  fflush (stdout);
305}

307static dds_entity_t prepare_dds(dds_entity_t *reader, const char *partitionName)
308{
309  dds_return_t status;
310  dds_entity_t topic;
311  dds_entity_t subscriber;
312  dds_listener_t *rd_listener;
313  dds_entity_t participant;
314
315  int32_t maxSamples = 4000;
316  const char *subParts[1];
317  dds_qos_t *subQos = dds_create_qos ();
318  dds_qos_t *tQos = dds_create_qos ();
319
320  /* A Participant is created for the default domain. */
321
322  participant = dds_create_participant (DDS_DOMAIN_DEFAULT, NULL, NULL);
323  if (participant < 0)
324    DDS_FATAL("dds_create_participant: %s\n", dds_strretcode(-participant));
325
326  /* A Topic is created for our sample type on the domain participant. */
327
328  dds_qset_reliability (tQos, DDS_RELIABILITY_RELIABLE, DDS_SECS (10));
329  dds_qset_history (tQos, DDS_HISTORY_KEEP_ALL, 0);
330  dds_qset_resource_limits (tQos, maxSamples, DDS_LENGTH_UNLIMITED, DDS_LENGTH_UNLIMITED);
331  topic = dds_create_topic (participant, &ThroughputModule_DataType_desc, "Throughput", tQos, NULL);
332  if (topic < 0)
333    DDS_FATAL("dds_create_topic: %s\n", dds_strretcode(-topic));
334
335  /* A Subscriber is created on the domain participant. */
336
337  subParts[0] = partitionName;
338  dds_qset_partition (subQos, 1, subParts);
339  subscriber = dds_create_subscriber (participant, subQos, NULL);
340  if (subscriber < 0)
341    DDS_FATAL("dds_create_subscriber: %s\n", dds_strretcode(-subscriber));
342  dds_delete_qos (subQos);
343
344  /* A Reader is created on the Subscriber & Topic with a modified Qos. */
345
346  rd_listener = dds_create_listener(NULL);
347  dds_lset_data_available(rd_listener, data_available_handler);
348
349  /* A Read Condition is created which is triggered when data is available to read */
350  waitSet = dds_create_waitset (participant);
351  if (waitSet < 0)
352    DDS_FATAL("dds_create_waitset: %s\n", dds_strretcode(-waitSet));
353
354  status = dds_waitset_attach (waitSet, waitSet, waitSet);
355  if (status < 0)
356    DDS_FATAL("dds_waitset_attach: %s\n", dds_strretcode(-status));
357
358  imap = HandleMap__alloc ();
359
360  memset (data, 0, sizeof (data));
361  for (unsigned int i = 0; i < MAX_SAMPLES; i++)
362  {
363    samples[i] = &data[i];
364  }
365
366  *reader = dds_create_reader (subscriber, topic, NULL, pollingDelay < 0 ? rd_listener : NULL);
367  if (*reader < 0)
368    DDS_FATAL("dds_create_reader: %s\n", dds_strretcode(-*reader));
369
370  if (pollingDelay == 0)
371  {
372    status = dds_waitset_attach (waitSet, *reader, *reader);
373    if (status < 0)
374      DDS_FATAL("dds_waitset_attach: %s\n", dds_strretcode(-status));
375  }
376
377  dds_delete_qos (tQos);
378  dds_delete_listener(rd_listener);
379
380  return participant;
381}
382
383static void finalize_dds(dds_entity_t participant)
384{
385  dds_return_t status;
386
387  for (unsigned int i = 0; i < MAX_SAMPLES; i++)
388  {
389    ThroughputModule_DataType_free (&data[i], DDS_FREE_CONTENTS);
390  }
391
392  status = dds_waitset_detach (waitSet, waitSet);
393  if (status < 0)
394    DDS_FATAL("dds_waitset_detach: %s\n", dds_strretcode(-status));
395  status = dds_delete (waitSet);
396  if (status < 0)
397    DDS_FATAL("dds_delete: %s\n", dds_strretcode(-status));
398  status = dds_delete (participant);
399  if (status < 0)
400    DDS_FATAL("dds_delete: %s\n", dds_strretcode(-status));
401}
