Linux Installation
==================

There are two ways to install |var-project| on Linux: using linux artifact or the debian package.

Linux Artifact
--------------

1. Download the |var-project| Linux artifact
2. Extract the downloaded archive
3. Run the executable file `./'CycloneDDS Insight'` located in the extracted folder


Debian Package
--------------

1. Download the |var-project| debian package (.deb file)
2. Install the package using the command:

.. code:: bash

  sudo apt install ./<package_name>.deb

3. After installation, you can run |var-project| from debians application menu or by executing `'CycloneDDS Insight'` from any terminal.
